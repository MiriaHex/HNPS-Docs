### 切换检查模式

检查模式开启时，可方便的查看所选方块的操作信息

```
/co inspect
```

### 查找方块记录

```
/co lookup u:<user> t:<time> r:<radius> a:<action> i:<include> e:<exclude>
```

{% hint style="info" %}
你可以在 CoreProtect官方文档中 找到各个参数的用法
{% endhint %}
