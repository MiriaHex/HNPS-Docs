# AuthMe

## 初次使用

使用以下命令进行 **注册**

```
/register [密码] [再输入一遍密码]
```

## 登录

使用以下命令进行 **登录**

```
/login [密码]
```

{% hint style="info" %}
在 5分钟 内重新连接服务器可以免去登录步骤
{% endhint %}

## 绑定邮箱

{% hint style="danger" %}
**我们强烈建议你绑定邮箱，这将是你找回密码的唯一方式**
{% endhint %}

使用以下命令进行 **绑定邮箱**

```
/email [邮箱账号] [再输入一遍邮箱账号]
```
