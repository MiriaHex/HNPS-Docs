# Geyser

## 命令

{% hint style="info" %}
此处仅展示在本服务器上可用的命令，如需完整文档，请参阅：

[Geyser的官方Wiki](https://wiki.geysermc.org/)
{% endhint %}

### 切换副手

```
/geyser offhand
```

### 打开 Java成就页面

```
/geyser advancement
```

### 打开 Java统计信息

```
/geyser statistics
```

### 打开 高亮信息显示

```
/geyser tooltips
```
