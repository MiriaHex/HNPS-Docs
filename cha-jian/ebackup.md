# eBackup

## 命令

{% hint style="info" %}
此处仅列出在本服务器上可用的命令，如需完整文档，请参阅：

[eBackup的Spigot页面](https://www.spigotmc.org/resources/ebackup-simple-and-reliable-backups-for-your-server-supports-ftp-sftp.69917/)
{% endhint %}

### 全局备份

```
/ebackup backuplocal
```

### 存储信息

```
/ebackup stats
```

### 备份列表

```
/ebackup list
```
